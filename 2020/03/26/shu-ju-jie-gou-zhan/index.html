限定性线性表——栈和队列

限定线性表运算的位置(只允许在表的端点)

栈

栈的定义
  1. 栈顶：表允许运算的一端
  2. 栈底： 不允许运算的一端
  3. 空栈：栈中没有元素
栈的常见运算
  1. 表尾插入(进栈)
  2. 表尾删除（出栈）
栈的特性
  1. 后进先出(LIFO)
栈的抽象数据类型定义
  1. 数据元素：属于同一数据对象的任意类型数据
  2. 关系：线性关系
  3. 基本操作：
    • InitStack(S) //初始化栈
    • ClearStack(S) //清空栈
    • IsEmpty(S) //判空
    • IsFull(S) //判满
    • Push(S,x) //进栈
    • Pop0(S,x) //出栈
    • GetTop(S,x) //读栈顶

栈运算的实现

栈在计算机中有两种存储方式：顺序存储和链式存储。

顺序栈
  1. 定义：用一组连续的存储单元一次存放自栈底到栈顶的数据元素
  2. 设一个栈顶指针top动态指示栈顶元素在顺序表中的位置
  3. top = -1 表示空栈
函数的实现
  1. 定义一个栈
#define True 1
#define False 0
#define Stack_Size 50
typedef struct
{
    StackElementType elem[Stack_Size];
    int top;
}SeqStack;
  1. 初始化
void InitStack(SeqStack *S)
{
    /*构造一个空栈*/
    S->top = -1;
}
//S是具有顺序栈类型的一个变量
  1. 判空
int IsEmpty(SeqStack *S)
{
    /* S为空返回true；S为假返回false */
    return (S->top == -1 ? TRUE:FALSE);
}
  1. 判满
int IsFull(SeqStack *S)
{
    return (S->top == Stack_Size ? TRUE:FALSE);
}
  1. 进栈
int Push(SeqStack *S,StackElementType x)
{
    if(S->top==Stack_Size) return False; //栈满
    S->top++;
    S->elem[S->top] = x;
    return TRUE;
}
  1. 出栈
int Pop(SepStack *S,StackElementType *x)
{
    if(S->top == -1) return False;
    else
    {
        *x = S->elem[S->top];
        S->top--;
        return TRUE;
    }
}
